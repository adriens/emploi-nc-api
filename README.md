# API REST : Offres d'emploi (https://emploi.gouv.nc/)
API Offres d'emploi de Nouvelle-Calédonie

## Démarrer le service
 
`mvn spring-boot:run`

## Endpoints

`/emploi/latest/{quantity}`

## Exemples d'appels

```
/emploi/latest
/emploi/latest/5
```
 
 Bientot disponible sur [Marketplace]().
